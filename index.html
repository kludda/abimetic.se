Projects

Gyroid mesh generator written in Python

Generates a mesh representing a Schoen gyroid, which is is an infinitely connected triply periodic minimal surface discovered by Alan Schoen in 1970.

The exiting thing with the gyroid is the high surface area and turbulent flow which makes it suitable for heat exchanger applications.

Currently, parametric CAD softwares aren't suitable to draw these kind of surfaces so they need to be created as mesh.

https://github.com/kludda/TPMS

Home energy system simulation tool

This CLI python tool generates a PDF report of a simulated home energy system with PV and battery storage.

I built it for myself to make an educated decision weather to invest in PV+battery or not, and what the properties of that system should be.

https://github.com/kludda/home_es_sim

A 3kWh/3kW 12VDC native DIY portable power station

A DIY 3kWh/3kW 12VDC native portable power station made for a camper van. It is power by four LiFePO4 280 Ah cells, have a BMS bypass with retained safety, modular design for on-the-road repairs.

https://github.com/kludda/portable-power-station
